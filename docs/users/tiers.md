# Gestion des tiers

Un **tiers** est toute entreprise ou personne avec laquelle vous avez une relation commerciale : un client à qui vous facturez, un fournisseur à qui vous achetez, un prospect à qui vous adressez un devis. Dolipocket gère les deux rôles dans la même fiche : un même tiers peut être à la fois client et fournisseur.

## Accéder à la liste des tiers

Depuis le menu principal, touchez **Tiers**. La page affiche la liste de tous vos tiers, triée par date de création (les plus récents en haut).

Au-dessus de la liste, une **barre de recherche** vous permet de filtrer en saisissant un nom, une partie de nom, un code client ou un code fournisseur. Le filtrage se fait au fil de la frappe.

Trois filtres rapides sont disponibles :

- **Tous** -- affiche tous les tiers
- **Clients** -- uniquement les tiers marqués comme client
- **Fournisseurs** -- uniquement les tiers marqués comme fournisseur

## Créer un nouveau tiers

Touchez le bouton **Nouveau tiers** (icône plus en haut à droite de la liste). Le formulaire vous demande :

- **Nom** -- raison sociale ou nom du contact (obligatoire)
- **Type** -- choisissez si ce tiers est client, prospect, fournisseur, ou plusieurs à la fois
- **Adresse, code postal, ville, pays** -- coordonnées postales
- **Téléphone, email, site web** -- coordonnées de contact
- **SIRET, code TVA, code APE** -- informations légales pour les sociétés françaises
- **Statut** -- actif ou désactivé (un tiers désactivé reste consultable mais n'apparaît plus dans les listes de sélection)

Validez avec **Enregistrer**. Le tiers est créé et la fiche s'ouvre, vous permettant immédiatement d'ajouter des contacts, des notes ou des documents.

## Consulter et modifier un tiers

Touchez une ligne de la liste pour ouvrir la fiche détaillée. Vous y voyez :

- Les coordonnées complètes
- Les contacts liés (voir [Contacts](contacts.md))
- L'historique des devis, commandes et factures associés
- Les documents joints

Pour modifier la fiche, touchez l'icône **Modifier** (crayon) en haut à droite. Le formulaire de saisie réapparaît avec les valeurs actuelles. Modifiez ce qui doit l'être et validez.

## Supprimer un tiers

Depuis la fiche d'un tiers, touchez le menu d'actions (les trois points en haut à droite) puis **Supprimer**. Une confirmation vous est demandée.

> **Attention :** la suppression n'est possible que si le tiers n'est lié à aucun document (devis, commande, facture). Sinon, le système affiche un message d'erreur. Pour cesser d'utiliser un tiers sans pouvoir le supprimer, désactivez-le en passant son statut à **Désactivé** : il disparaîtra des listes de sélection mais l'historique reste consultable.

## Bonnes pratiques

- **Renseignez le SIRET et la TVA** dès la création pour les sociétés françaises : ces champs apparaîtront automatiquement sur les factures
- **Utilisez les notes** pour mémoriser les conditions particulières (délai de paiement négocié, contact privilégié, langue de communication)
- **Vérifiez les doublons** : avant de créer un tiers, recherchez son nom dans la liste pour éviter de saisir deux fois le même client
- **Ajoutez plusieurs contacts** pour les structures avec plusieurs interlocuteurs (commercial, comptable, technique) plutôt que de créer plusieurs fiches tiers
